## HOWTO'S

Run development server:
```
npm run server
```

Build assets for production environment:
```
npm run build:prod
```

Build assets for development environment:
```
npm run build:dev
```

Run production server for serving statics:
```
npm run server:prod
```

### Scripts heroku related:
```
npm run postinstall
npm run start
```
